## Overview

- Built a real-time analytics dashboard that visualizes operational metrics and system health for high-traffic applications. The goal was to provide developers and product teams with a single interface to monitor performance, identify anomalies, and respond to issues faster.
- The project focuses on building a responsive and data-intensive frontend using React and modern state management patterns.

## Key Features

- **Real-time metrics streaming**: Built a set of modular, reusable UI components using React and styled-components in Next.js, focusing on accessibility and responsiveness.
- **Interactive charts**: Integrated a theming system that allows easy switching and customization of color palettes, typography, and layout styles using CSS variables and Figma tokens.
- **Filtering & Time range selection**: Collaborated closely with designers by setting up a shared design library in Figma. This library was synchronized with the codebase, ensuring design handoffs were seamless and that design tokens remained consistent across both platforms.
- **Dynamic Dashboards**: Developed comprehensive documentation with Storybook to showcase components, usage patterns, and best practices, ensuring the design system is easy to adopt by other teams.

## Challenges and Learnings

One of the main challenges was handling high-frequency data updates without causing performance issues in the UI. Early versions of the dashboard re-rendered too frequently, resulting in noticeable lag during heavy metric streams.

To address this, I introduced memoized components and selective state updates, ensuring only the components receiving new data were re-rendered. I also optimized chart rendering by batching updates and limiting the refresh interval.

This project strengthened my understanding of:
-performance optimization in React applications
-managing real-time data streams
-designing reusable visualization components

## Outcomes

- Reduced unnecessary component re-renders by ~40%, significantly improving UI responsiveness.
- Built 10+ reusable visualization components used across multiple dashboard views.
- Achieved smooth real-time updates with sub-second latency
- Delivered a scalable dashboard architecture capable of handling thousands of metric updates per minute
