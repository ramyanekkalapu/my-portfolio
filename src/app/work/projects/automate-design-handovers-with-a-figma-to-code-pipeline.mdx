## Overview

Developed a workflow management interface designed to streamline task coordination across teams. The application enables users to create, assign, and track tasks while maintaining visibility into project progress.

The focus of this project was building a modular Angular application that mirrors real enterprise UI systems.

## Key Features

- **Role-based Dashboards**: Developed a custom Figma plugin that extracts design tokens such as colors, typography, and spacing values, and exports them in a format compatible with our codebase.
- **Dynamic task boards**: Integrated an automated process that translates Figma components into React code using a combination of design tokens and pre-built component templates. This allowed developers to focus more on logic and less on repetitive UI coding.
- **Reactive Forms**: Established a CI/CD pipeline that continuously synchronizes design changes from Figma to the codebase, ensuring design updates are reflected instantly without manual intervention.
- **Real-time workflow updates**: Leveraged a design system that remains the single source of truth for both designers and developers, making it easy to maintain consistency even as the product evolves.

## Challenges and Learnings

A key challenge was managing complex asynchronous workflows while keeping the UI predictable and responsive.

To handle this, I used RxJS streams to centralize state updates and avoid inconsistent UI states caused by multiple concurrent API requests. I also implemented shared services to manage data flow between components.

This project helped deepen my understanding of:

- Angular reactive programming patterns
- scalable frontend architecture in enterprise environments
- managing complex application state

## Outcome

- Developed 20+ reusable UI components, improving development efficiency and maintainability

- Reduced duplicated UI logic by ~30% through shared services and modular architecture

- Improved API interaction reliability by implementing structured error handling and retry logic

- Delivered a fully functional workflow UI capable of managing hundreds of tasks across multiple users
